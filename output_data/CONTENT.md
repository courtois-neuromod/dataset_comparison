# Output Data

Generated by `invoke run`:

- `datasets_tidy.csv` — tidy long-format table produced by `make_tables`; columns: `dataset`, `group`, `group_color`, `modality`, `dotpath`, `unit`, `value`. One row per (dataset × modality) combination.
- `*.png` — figures produced by `summary.ipynb`, which reads `datasets_tidy.csv`

See `source_data/CONTENT.md` for inputs.
